# 📺 复古频道电视

把 B站、YouTube 视频排成电视频道，像拨老电视一样换台，配上 CRT 扫描线和 VHS 噪点滤镜。

![后端](https://img.shields.io/badge/后端-Python%203-yellow)

## ✨ 功能

- **多频道排播** — 每个频道有独立的节目表，按全局时间轴滚动播放，多端同步进度
- **支持 B站 / YouTube / 直连视频 / 任意 iframe**
- **换台动画** — 带静噪闪烁效果，模拟老电视换台手感
- **画面滤镜** — 原始 / 老电视(CRT) / 录像带(VHS) / 黑白 / 霓虹 / 绿屏 六种风格
- **节目预告** — 右侧面板显示当前频道节目表和播放进度
- **搜片** — 输入关键词自动从 B站、YouTube 搜索并加入节目表
- **批量导入** — 支持 B站番剧 / 合集 / 多 P 视频一键导入
- **频道管理** — 后台新增、编辑、删除频道，多端实时同步（SQLite）
- **键盘操控** — 方向键换台，`F` 切换滤镜，`H` 显示/隐藏后台面板

---

## 🚀 快速开始

**依赖：Python 3.6+，无需额外安装第三方库**

```bash
# 克隆仓库
git clone https://github.com/NoharaHiroshi/retro-channel-tv.git
cd retro-channel-tv

# 启动服务
python3 server.py

# 浏览器打开
# http://localhost:8017
```

局域网内其他设备也可以用你的 IP 访问同一个地址，共享同一份频道数据库。

## ⌨️ 操作方式

| 操作 | 功能 |
|------|------|
| `↑` / `↓` | 换台 |
| 数字键 `1–9` | 直接跳转对应频道 |
| `F` | 循环切换画面滤镜 |
| `H` | 显示 / 隐藏后台管理面板 |
| `Enter` | 开始播放（首次进入需点击） |
| 点击频道列表 | 直接跳台 |

## 📡 添加节目

在右侧面板底部的输入框里，可以：

- **搜索关键词** — 自动从 B站和 YouTube 搜索并加入当前频道
- **粘贴链接** — 支持以下格式直接加入：
  - B站视频链接 / BV 号（如 `BV1xxxxxxxxxxxx`）
  - YouTube 链接 / 视频 ID
  - `.mp4` / `.webm` / `.m3u8` 直链
  - 任意可嵌入的网页地址
- **批量导入 B站** — 粘贴番剧 `ss/ep`、合集 `lists`、多 P 视频链接，点击「B站导入」

## 🧩 Chrome 插件增强（推荐）

> 安装插件后，**B站视频换台时控制栏会自动隐藏**，不再需要手动移入移出鼠标，电视沉浸感大幅提升。

### 安装步骤

1. 打开 Chrome，地址栏输入 `chrome://extensions`
2. 右上角开启 **开发者模式**
3. 点击 **加载已解压的扩展程序**
4. 选择项目目录下的 `bili-ctrl-hide` 文件夹

### 插件行为

- **换台后** — 控制栏自动隐藏，保持干净的全屏画面
- **鼠标移入播放器** — 停止干预，B站原生控制栏完全恢复，可以正常操作进度条、音量等
- **鼠标移出** — 下次换台时再次自动隐藏
- **不影响直接访问** `player.bilibili.com` — 插件只在嵌入 iframe 时生效

## 📱 移动端

手机浏览器可以正常访问和换台。B站视频在移动端会以点击播放方式加载（移动浏览器限制自动播放），右下角提供「B站」直达按钮，可在 B站 App 中打开当前视频。

## 🛠️ 部署到服务器

```bash
# 指定端口启动（默认 8017）
python3 server.py 8017

# 后台持续运行
nohup python3 server.py > tv.log 2>&1 &
```

需在服务器防火墙（云服务器还需在控制台安全组）开放对应端口的 TCP 入站规则。
